Fișa de opțiuni recomandată

Ordinea = preferință strict descrescătoare; algoritmul dă primul cod cu locuri libere, deci visurile puse primele nu costă nimic. Rândurile punctate sunt decizii de familie (compromisuri de profil) — dacă le scoateți, numerotarea se decalează. Procentele se recalculează după ipotezele de mai sus.
